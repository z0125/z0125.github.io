什么是Hexo？

4.生成SSH-KEY并添加到github

1.检查是否生成过SSH key

检查是否已经存在 id_rsa.pub 或 id_dsa.pub 文件，如果文件已经存在，那么你可以跳过步骤2，直接进入步骤3。

2.创建SSH key

  • 提示输入文件名，直接回车使用默认文件名（推荐），生成 id_rsa 和 id_rsa.pub 两个秘钥文件。
  • 接着又会提示你输入两次密码（该密码是你push文件的时候要输入的密码，而不是github管理者的密码），
    可以不输入密码，直接按回车。push的时候就不需要输入密码，直接提交到github上。

3.配置SSH key到github上

  1. 拷贝 id_rsa.pub 文件的内容（RSA公钥）
  2. 登录github账号，从右上角的Settings进入，点击左侧菜单栏的 SSH and GPG keys 进入添加 SSH key。
  3. 把拷贝的公钥串粘贴到 key 所对应的输入框中，注意公钥串前后不要有空格或者回车

4.测试SSH key 是否配置成功

  • 提示授权警告信息时键入yes
  • 如果创建SSH key 时有输入密码，会要求输入密码验证
  • 返回successfully等消息提示，说明SSH key已经成功设置并生效
